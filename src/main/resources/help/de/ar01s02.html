2. RTeasy starten

Um RTeasy benutzen zu können benötigen sie eine Java Virtual Machine(JVM). Bei Problemen überprüfen Sie bitte die Systemvoraussetzungen auf den zugehörigen Webseiten des Instituts für Technische Informatik an der Universität zu Lübeck.

Sie starten RTeasy über die Shell mit dem Befehl java -jar rteasy_<version>.jar

Es öffnet sich das Hauptfenster (Abschnitt 3: Hauptfenster) . Sie können nun RTeasy die Einstellungen von RTeasy anpassen (Einstellungen ), ein neues Programm erstellen oder ein vorhandenes Programm öffnen (Abschnitt 3.1.1.1: Dateimenü)
