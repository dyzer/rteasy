8. Breakpoint Window

This window shows all breakpoints where the running simulation can stop.

Add .  adds a new breakpoint. First click the button Add then click the statement sequence in the editor window where the simulation should stop. The state will be added to the table in the breakpoint window and the corrseponding statements will be highlighted in the appropriate color (Section 4.2: In the Simualtion Mode).

Delete .  deletes the selected breakpoint.

Release Marking .  releases the highlighting of the corresponding breakpoint.
