2. How to start RTeasy

Rteasy requires a Java Virtual Maschine. When arising problems please check the system requirements at the websites of the Institute of Computer Engineering at the University of Luebeck.

You start RTeays by using the following shell command java -jar rteasy_<version>.jar

In the appearing main window (Section 3: Main Window) you can change the settings (Settings ), start up a new project or open an existing project (Section 3.1.1.1: File Menu;)
