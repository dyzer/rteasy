5. Simulation Window

The simulation window is only visible during a simulation (Section 3.1.2: Simulation Mode). It displays the information of all declared registers, busses, memories and register arrays and the current state and cycle.

State Panel. 

This panel contains the current state and cycle.

Registers and Busses. 

This table displays the current entries of all declared components. A change of an entry is highlighted in yellow. By double-clicking a table cell that displays an entry it becomes editable an a new entry can be entered. To aplly the changes use the enter key. To discard them use the escape key.

Tabelle 3. Base Table

Wert Base Characters Description
BIN20, 1binary output
DEC100,...,9 decimal output - unsigned
HEX160,..,9,A,...,Fhexadecimal output - unsigned
DEC2100,..,9decimal output - signed
HEX2160,...,9,A,...,Fhexadecimal output - signed

Entries that are to wide for the corresponding register/bus will be cut to the needed width.

Memory. 

Each declared memory displays a button that opens the window where the content of the corresponding memory is shown and editable (Section 6: Memory).

Registerarrays. 

Each declared register array has this button as well to manage ist contents. (Section 7: Registerarrays).
