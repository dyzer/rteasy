4. Editor Window

The editor shows the source code of the current project. In the edit mode (Section 3.1.1: Edit Mode) you can process on the code. The simulation mode (Section 3.1.2: Simulation Mode) shows the current simulation state in the editor window. Note that the simulation mode disables any processing in the editor.

4.1. In the Edit Mode

The editor provides all common functions.

Mark, Cut, Copy and Paste.  These functions are available as usual. The following table shows the shortcuts.
